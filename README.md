Task 4 — NGO Website Feature Highlights 🎨💻

InAmigos Foundation — Web Development Internship 2026


📌 Task Overview
Design a layout highlighting important features that should be present in an NGO website and can be implemented in InAmigos Foundation's website.
Objective: Identify essential NGO website features and present them visually with short labels and notes explaining each feature.

🌿 Live Preview
🔗 Click here to view the design

✨ Features Designed (5 Key Features)
💳 Feature 1 — Smart Donation Section

Preset donation amounts (₹100, ₹500, ₹1K, ₹5K)
Custom amount input
One-time / Monthly / Yearly frequency toggle
Real-time impact visualization (what your money does)
UPI, Card, Net Banking payment methods
80G Tax Exempt badge for donor trust


🤝 Feature 2 — Volunteer Registration Form

Clean two-column layout with motivation panel + form
Skill-based matching (Web Design, Teaching, Field Work, etc.)
Project selection dropdown (all 6 InAmigos projects)
Displays live volunteer count — 200+ across 28 states
Converts curiosity into commitment


📊 Feature 3 — Impact Showcase

4 key stats: 50K+ Beneficiaries, 200+ Volunteers, 5K+ Trees, 28 States
Journey timeline from 2020 founding to 2026 goals
State-wise volunteer distribution map
Counter animations on scroll for engagement


💬 Feature 4 — Testimonials Section

Featured story card (volunteer who became core team)
Regular volunteer and beneficiary quotes
Star ratings for credibility
Video testimonial card (Founder's story)
Mix of voices: interns, volunteers, beneficiaries, parents


🎯 Feature 5 — Live Campaign Cards

Campaign cards with progress bars and fundraising goals
Urgency badges (Urgent / Active / New)
Donor count and days remaining for FOMO effect
Mini cards for all 6 projects (Seva, Bachpanshala, Prakriti, Udaan, Jeev, Vikas)
"Start a Campaign" CTA for peer-to-peer fundraising
